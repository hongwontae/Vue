1. npm run build
=> 이 빌드는 개발 모드에서 작성한 소스 코드를 최적화된 형태로 변환하여 배포할 준비를 합니다.


2. 단계
2-1. 소스 코드 압축 및 최적화
=> webpack를 통해 JS 코드 최소화, 트랜스파일링하여 구형 브라우저에서도 호환되게 합니다.
=> Vue 파일 내의 템플릿, 스크립트, 스타일을 최적화합니다.
=> 모든 CSS 파일을 합치고, 불필요한 스타일을 제거하여 파일 크기를 줄입니다.


2-2. 파일 분할
=> Vue-Router에서는 페이지별로 코드 분할을 설정할 수 있습니다.
=> 페이지가 로드될 때 해당 페이지에 필요한 JavaScript만 로딩되게 합니다.

=> 작은 컴포넌트를 코드 분할하여, 필요할 때만 로딩합니다.


2-3. 이미지 및 기타 자원 최적화
=> 프로젝트에 포함된 이미지 파일들을 압축하고, 적절한 포맷으로 변환하여 최적화합니다.


2-4. 서비스 워커 및 PWA
=> Vue CLI에서 PWA 플러그인을 활성화하면 서비스 워커가 생성되어 오프라인 지원을 할 수 있습니다.


2-5. 배포 준비
=> 배포 가능한 빌드 파일 생성: dist/ 폴더에 최적화된 정적 파일들이 생성됩니다.
=> 이 폴더 내에는 HTML, JS, CSS, 이미지 파일들이 들어 있으며, 
 이를 웹 서버에 배포하여 웹 앱을 제공할 수 있습니다.

1. v-if
=> 조건에 따라 DOM 요소를 렌더링하거나 제거하는 데 사용됩니다.
=> v-if는 특정 조건이 참일 때만 해당 요소를 렌더링하며, 조건이 거짓일 때는 해당 요소를 DOM에서 제거합니다.

1-1. v-else
=>  v-if나 v-else-if가 false일 때 렌더링됩니다.

1-2. v-else-if
=> 여러 조건을 처리할 때 사용됩니다.
=> v-if가 false일 경우, v-else-if 조건이 참이면 그에 해당하는 DOM을 렌더링합니다.

1-3. v-if가 여러개
=> v-if가 여러개이고 true가 되었을 때 그 뒤의 v-if를 평가하지 않습니다.
=> v-if가 여러개 이고 false가 되었을 떄 true를 찾습니다.

** 일반 JS 개념 (갑자기 헷갈렸다.)


2. v-for
=> 배열 또는 객체를 순회하면서 데이터를 기반으로 DOM 요소를 반복적으로 렌더링하는 데 사용됩니다.
=> 각 요소에 key 속성을 부여하여 DOM 업데이트를 효율적으로 수행할 수 있습니다.

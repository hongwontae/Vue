1. PWA (Progressive Web App)
=> 웹 앱을 네이티브 앱처럼 사용할 수 있도록 만들어주는 기술입니다.
=> 사용자 경험을 모바일 앱처럼 개선하려는 목적입니다.


2. Service Worker
=> PWA의 핵심 요소 중 하나로, 브라우저와 서버 사이에서 백그라운드에서 실행되는 스크립트입니다. 
=> 네트워크가 없거나 불안정한 환경에서도 앱을 사용할 수 있도록 도와줍니다.
=> 오프라인 지원, 푸시 알림, 백그라운드 데이터 동기화...


3. PWA 플러그인
=> Vue CLI나 React 같은 프레임워크에서 PWA 플러그인은 앱을 PWA로 변환해주는 도구입니다.
=> 서비스 워커와 같은 PWA 관련 기능을 쉽게 설정할 수 있습니다.

1. createHistory()
=> Vue Router에서 URL의 히스토리 모드를 설정하는 데 사용됩니다.
=> URL에서 해시(#)를 제거하고, 일반적인 URL처럼 작동하도록 설정할 수 있습니다. 
=> 페이지를 새로 고침할 때도 올바른 URL 경로를 유지할 수 있습니다.


2. NotFound 처리
=> /:path(.*)*
=> 동적 매개변수
=> (.*), 정규식으로 0개 이상의 모든 문자을 의미합니다. 해당 경로 뒤에 오는 어떤 문자든지 모두 캡쳐합니다.
=> *, Vue Router에서 path 경로 뒤에 오는 모든 부분을 포함하게 만듭니다.
=> ':동적매개변수(.*)*'를 사용하는것이 이상적입니다.

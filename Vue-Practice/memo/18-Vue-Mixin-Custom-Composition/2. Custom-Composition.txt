1. Custom Composition API
=> Composition API 재사용 로직을 설계하는 문법입니다.
=> setup(){}에서 함수를 호출하여 재사용을 가능케합니다.
=> 함수 이름은 앞에 use가 붙어야 합니다. 재사용 로직을 전개합니다.
=> setup에서 use로 정의한 커스텀 컴포지션을 import하여 호출하고 값을 받습니다.
=> 해당 값을 반환합니다. => 재사용 로직을 사용할 수 있습니다.


2. 커스텀 컴포지션 API도 기존의 setup의 로직과 병합됩니다.
=> 함수의 return 값을 사용하기에 충돌하지 않습니다.

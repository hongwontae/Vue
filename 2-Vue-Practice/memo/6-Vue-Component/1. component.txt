1. Vue v-for로 동적으로 렌더링
=> 컴포넌트를 사용하지 않고 v-for를 사용해서 동적인 데이터를 표현했을 떄
동적인 데이터를 식별하지 못해서 원치 않는 반응을 보일 수 있습니다.
=> 또한 코드를 유지보수하기에 어렵습니다.


2. Vue Component
=> createApp의 반환값의 메서드를 통해 컴포넌트를 생성할 수 있습니다.
=> app.component('커스텀 HTML 요소 이름', 설정 객체);
=> 커스텀 HTML 요소 이름은 이름이 두 개이상, - 로 연결되어야 합니다. (기존 HTML과 충돌을 방지하기 위해서)
=> createApp의 설정 객체와 유사합니다. data, methods...

*** 만약 Vue Component을 설정하고 사용한다면 같은 컴포넌트라도 서로 독립적입니다. ***
=> 이 사항이 제일 중요합니다.
